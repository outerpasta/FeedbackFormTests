## Synopsis

Tests for Vacume Cleaner & Anvil Co. Feedback Form

## Installation

Requires python 2.7!

Install python dependencies:
```
$ cd FeedbackFormTests/
$ pip install -r requirements.txt
```

## Running

Run test for Firefox:
```
$ pybot form_tests/
```

Run test for each browser:
```
$ sh bin/allbrowsers.sh
```
